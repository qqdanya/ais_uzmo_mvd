# Финальная проверка перед сдачей

## 1. Команды проверки

```bash
python manage.py makemigrations --check
python manage.py migrate
python manage.py check
python manage.py test
```

## 2. Проверка интерфейса

- Вход под руководителем.
- Открытие главного экрана.
- Выбор территориального органа.
- Создание заявки в каждом ключевом отделе.
- Редактирование заявки.
- Изменение статуса заявки.
- Проверка истории статусов.
- Прикрепление фотографий к заявке.
- Просмотр фотографий через lightbox.
- Создание папки фотографий.
- Загрузка нескольких фотографий.
- Скачивание фотографии, папки и общего архива.

## 3. Проверка административной панели

- Оперативная сводка открывается без ошибок.
- Фильтры по датам и территориальным органам работают.
- Раздел «Заявки» фильтрует по органам, отделам, статусам и поиску.
- Раздел «Территориальные органы» показывает корректные показатели.
- Раздел «Отделы» показывает корректные показатели.
- Раздел «Материальная база» показывает актуальный срез.
- Раздел «Сотрудники» показывает онлайн-статусы без перезагрузки.
- Создание сотрудника работает.
- Редактирование прав сотрудника работает.
- Блокировка и разблокировка сотрудника работают.
- Сброс активации работает.
- Журнал действий открывается и фильтруется.
- Настройки порогов сохраняются.

## 4. Проверка прав доступа

- Оператор видит только доступные органы и отделы.
- Наблюдатель не может изменять данные.
- Обычный сотрудник не видит административную панель без прав.
- Руководитель имеет полный доступ.
- Руководитель не может случайно заблокировать сам себя через карточку сотрудника.
- Доступ к таблицам БД есть только у руководителя.

## 5. Проверка номеров заявок

- Нельзя создать две заявки с одинаковым номером в одном территориальном органе и одном отделе.
- Можно создать одинаковый номер в разных отделах.
- Можно создать одинаковый номер в разных территориальных органах.
- При редактировании заявки конфликт номера показывается понятным сообщением.

## 6. Проверка перед коммитом

```bash
git status
git diff --stat
```

В репозиторий не должны попасть:

- `.env`
- `db.sqlite3`
- `media/`
- `staticfiles/`
- `dashboard_thresholds.json`
- `__pycache__/`
